# Discord-Bot
A feature-rich Discord RPG bot built with Python, featuring classes, combat, dungeons, bosses, quests, inventory, equipment, guilds, and a persistent MongoDB database.
🏰 Discord RPG Bot

A modern and feature-rich Discord RPG bot built with Python, discord.py, and MongoDB. Create your character, choose a class, battle monsters, conquer dungeons, defeat powerful bosses, collect legendary equipment, and climb the global leaderboards.

---

✨ Features

- ⚔️ Turn-based combat system
- 🛡️ Multiple playable classes
- 📈 Player leveling and progression
- 🎒 Inventory and equipment system
- 💰 Economy with gold and trading
- 🏰 Guild creation and management
- 👹 World bosses and raids
- 🗺️ Procedurally generated dungeons
- 📜 Daily and story quests
- 🧪 Skills and status effects
- 🎁 Loot drops and rarity system
- 🏆 Global leaderboards
- 💾 Persistent MongoDB database
- ⚡ Fast asynchronous architecture

---

🛠️ Built With

- Python 3.11+
- discord.py
- MongoDB
- Motor
- Pillow
- aiohttp

🚀 Installation

Clone the repository:

git clone <repository-url>
cd BOT

Install dependencies:

pip install -r requirements.txt

Create a ".env" file:

TOKEN=YOUR_DISCORD_BOT_TOKEN
MONGO_URI=YOUR_MONGODB_CONNECTION_STRING

Run the bot:

python main.py
